# CanLib
Библиотека для работы с CAN-шиной.
--------------
В ней будут реализованы основные инструменты для отображения, записи и генерации трафика CAN.

№ | функции                           | готовность
--|-----------------------------------|------
1 | отправка пакетов                  | 
2 | получение пакетов                 | 
3 | генерация случайного пакета       | 
4 | запись данных в файл              |
5 | отображение пакетов               |
